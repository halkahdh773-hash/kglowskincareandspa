
Kglow Skincare and Spa - Website package
Files:
- index.html  --> main website
- images/kglow_logo.jpg  --> logo used in header & popup

How to publish on GitHub Pages (phone):
1. Open GitHub and create a new repository named 'kglowskincareandspa' (Public).
2. Upload index.html and the images folder (your-logo.png).
   - In GitHub app: New repository -> Upload files -> choose files from phone storage.
3. Commit changes.
4. In the repository on GitHub.com, go to Settings -> Pages -> choose branch 'main' and folder '/ (root)' -> Save.
5. Your site will be live at: https://yourusername.github.io/kglowskincareandspa
Notes:
- To change texts or prices: edit index.html and re-upload.
- The booking uses WhatsApp link; payments are manual (bank transfer).
